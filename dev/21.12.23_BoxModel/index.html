Box-sizing property

hilft uns Box Model zu manipulieren

box-sizing:content-box => default Wert, Boxen werden auf addiert

box-sizing:border-box => Breite wird automatisch mit Border verrechnet

Display property:inline, block inline-block

inline: Elemente können Höhe und Breite annehmen

block: Elemente nehmen 100% der Breite ein

inline-block: kombiniert das beste von Beidem, man hat nicht immer 100% Breite & man kann trotzdem eine Höhe und Breite vergeben.
